GitHub Pages 即時部署 · 前端單頁應用程式

你的網頁軟體，從這個模板開始。

這是一個適合放在 GitHub 上的前端模板：結構清楚、樣式現代、已預留 JavaScript 入口點。你可以把它改成股票追蹤系統、儀表板或任何你想做的工具。

前端技術棧
HTML · CSS · 原生 JavaScript
部署方式
GitHub Pages / 靜態主機
適合用途
儀表板 · 工具型網站 · Demo
應用狀態 · 開發模式
local / GitHub Pages

這裡可以放：

  • 假資料（例如：股票清單、使用者列表）
  • 目前登入狀態 / API 連線狀態
  • 系統日誌、錯誤訊息、除錯資訊

你可以在 script.js 中寫邏輯，透過 document.getElementById() 或 querySelector() 操作這個區塊。

前端載入完成，尚未連接任何 API。
